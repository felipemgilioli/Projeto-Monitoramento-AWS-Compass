Projeto de Monitoramento - Nginx + Linux

Servidor Web Local

Este projeto simula um ambiente de monitoramento com automação em Shell Script e servidor Nginx.

Objetivo do Projeto

O objetivo deste projeto é testar e demonstrar habilidades em Linux, automação com Shell Script, monitoramento e implantação de um servidor web local (Nginx) em uma VM Ubuntu na AWS.

Recursos Utilizados

  • Instância EC2 com Ubuntu Server
  • Servidor Nginx
  • Scripts Bash para monitoramento
  • Webhook para alertas (Discord)
  • Crontab para agendamento
